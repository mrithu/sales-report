## Features

### Smart Font Application
- Applies Kanit font to all text and headings
- Preserves readability and high energy aesthetic

### Text Styling
- Smart color selection based on light background (`#FFFFFF`)
- High contrast with black (`#000000`) text

### UI Components and Shapes
- Primary buttons are sharp (0px border radius) using the primary blue (`#72E2FF`).
- Secondary buttons feature rounded corners (6px border radius) and borders.
- Maintains visual interest while staying on-brand using the neon secondary colors.

## Technical Details

### Font Management
- Uses system-installed Kanit font when available
- For web artifacts, import Kanit from Google Fonts

### Color Application
- Uses precise Hex color values for precise brand matching (`#72E2FF`, `#CEFC17`, etc.)
