### Colors

**Theme**: Dark

**Core Colors:**
- Primary: `#FF6B35`
- Secondary: `#004E89`
- Accent: `#F77F00`

**UI Colors:**
- Background: `#1A1A1A`

### Typography

- **Primary / Heading Font**: Inter
- **Code Font**: Roboto Mono
- **Note**: Fonts should be pre-installed or imported in your environment for best results.

**Font Sizes:**
- H1: 48px
- H2: 36px
- H3: 24px
- Body: 16px

**Font Weights:**
- Regular: 400
- Medium: 500
- Bold: 700

## Features

### Smart Font Application
- Applies Inter font to all text and headings
- Applies Roboto Mono to code blocks
- Preserves readability across all systems

### Text Styling
- Uses precise font weights (400, 500, 700) for hierarchy
- Smart color selection based on dark background (`#1A1A1A`)
- Preserves text hierarchy with defined font sizes (48px to 16px)

### UI Components and Shapes
- Buttons use defined background colors, borders, and an 8px border radius.
- Maintains visual interest while staying on-brand using the primary (`#FF6B35`) and accent (`#F77F00`) colors.

## Technical Details

### Font Management
- Uses system-installed Inter and Roboto Mono fonts when available
- No font installation required - works with existing system fonts
- For web artifacts, import from Google Fonts if possible

### Color Application
- Uses precise Hex color values for precise brand matching (`#FF6B35`, `#004E89`, etc.)
- Maintains color fidelity across different systems
